# Personal Homepage

Static academic homepage inspired by a clean researcher-profile layout.

## Files

- `index.html`: page content and section structure
- `styles.css`: responsive layout and visual styling
- `assets/profile-placeholder.png`: generated placeholder portrait

## Edit

Replace the placeholder text in `index.html`:

- `Your Name`
- `Your Lab, Your University`
- email and profile links
- biography
- news
- publications
- experience and education

Replace `assets/profile-placeholder.png` with your own portrait when ready.

Open `index.html` directly in a browser to view the site.
